# Publicacao do Projeto Pinia

## URL esperada

```text
https://ericson-j-santos.github.io/projeto-pinia/
```

## Execucao local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Regras

- Nao versionar credenciais.
- Nao versionar arquivos `.env`.
- Nao publicar dados pessoais reais.
- Manter a aplicacao como demo estatica.
